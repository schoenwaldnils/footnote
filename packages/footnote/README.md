# Schoenwald footnote

> React component shows Schoenwald 'made by' footnote

## Install

```sh
npm install @schoenwald/footnote
```

## Usage

```ts
import { Footnote } from '@schoenwald/footnote'
```

use on light backgrounds

```ts
<Footnote />
```

use on dark backgrounds

```ts
<Footnote theme="dark" />
```
